# nextjs-typescript-starter

## create project

```bash
npx create-next-app app --example "https://github.com/redimpulz/nextjs-typescript-starter/tree/with_firebase"
```

## deploy

```bash
npm run deploy
```

## features

* Next.js
* TypeScript
* ESlint
* Prettier
* VScode Extentions
* Firebase Deployment
